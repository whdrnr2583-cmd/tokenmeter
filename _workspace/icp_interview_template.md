## Mom Test 5 질문 (15분 인터뷰)

### Q1. 최근 1개월 AI 도구에 얼마 썼는지 알고 계세요?
- 목적: 비용 가시화 통증 검증
- ✅ "정확히 모름" / "Max 플랜이라 신경 안 씀" → 통증 확인
- ⚠ "정확히 알아요. Anthropic Console에서 매일 확인" → 약한 ICP

### Q2. 어디서 토큰이 새는지 어떻게 확인하세요?
- 목적: MCP·도구별 분해 가치 검증
- ✅ "MCP 응답이 크다는 느낌만 있고 정확히 모름" / "ccusage 쓰는데 MCP 단위 안 보임" → 통증 확인
- ⚠ "잘 모르겠어요. 신경 안 써요" → 약한 ICP

### Q3. MCP 응답 토큰 양이 부담된 적 있나요?
- 목적: D-019 MCP 분해 차별점 검증
- 어떤 MCP가 제일 크나? Notion / GitHub / 자체 만든 거?
- "fields 화이트리스트" 시도해본 적 있나?

### Q4. (Token Meter 데모 1분 보여주고) 1주 써본다면 어디 봐주실 것 같나요?
- 목적: retention 메커니즘 발견
- "MCP 분해" / "비용 예측" / "세션 드릴다운" / 다른 것?
- **써보세요 / 결제 / 가격 절대 X**

### Q5. 무료 단계에서 어떤 기능이 부족하면 떠나실 것 같나요?
- 목적: Free → Pro 전환 트리거 검증
- 히스토리 7일 vs 30일?
- Smart alerts 1개 룰 vs 무제한?
- Webhook 연동?

- 결제 의향 직접 묻기 ("월 $5면 결제할 의향 있어요?") — **Mom Test 위반**. 거짓말 가능성 50%

## 응답 분류 매트릭스

| 분류 | 응답 패턴 | 해석 |
|---|---|---|
| ✅ Strong | "맞아요 그게 문제예요. 어디서 새는지 모르겠어요" + 구체 일화 | 페르소나 정합 + WTP 가능성 |
