# 01. 문제·시장

## 1.1 두 페르소나, 두 통증

### 페르소나 A — 클라우드 AI 파워유저
**누구**: Claude Pro/Max + Codex 동시 결제, 월 $100~500 AI 지출
**통증**:
1. 어디서 토큰이 새는지 모름 ("94% of developers get token tracking wrong")
2. 2개 클라이언트 각각 다른 대시보드 → 통합 뷰 없음
3. 알아도 줄일 방법 모름 (권장·자동화 도구 부재)
4. 모델별 성능·가성비 비교 어려움

### 페르소나 B — 로컬 LLM 빌더
**누구**: $1k+ GPU 보유, Ollama/LM Studio/llama.cpp/vLLM 사용
**통증**:
1. "내 GPU에서 어느 모델이 가장 빠른지" 모름 (8개 설치돼있는데)
2. 양자화·모델 변경 시 성능 차이 비교 도구 부족
3. "로컬로 처리한 게 클라우드면 얼마였는지" 모름 → ROI 계산 불가
4. 시간대·시스템 상태별 성능 변동 추적 도구 없음
5. CLI 벤치마크 도구는 단발성 — 지속 모니터링 부재

### 합치는 가치
대부분 시리어스 AI 빌더는 **클라우드 + 로컬 병행 사용**.
→ **"내 LLM 스택 전체 = 한 곳에서"** 라는 가치가 양쪽 모두에게 결제 정당화.

## 1.2 정량 증거

### 클라우드 측 (페르소나 A)
- 에이전트 토큰의 **67~70%가 낭비** (Medium 2026-04, ArXiv 2510.26585)
- 단일 HRIS API 응답 = 수십만 캐릭터 JSON
- GitHub MCP 스키마만 17,600 토큰
- "I Tracked Every Token for 30 Days — 94% Get It Wrong" (DEV)
- "StuckLoopDetection: Agent Burned $12 on 47 Identical Calls" (Medium)

### 로컬 측 (페르소나 B)
- Ollama RTX 4090 + 7B: 48~78 tok/s (모델·설정별 변동 큼)
- vLLM 187 tok/s vs Ollama 82 tok/s (동일 모델 다른 엔진) — **3배 차이**
- LM Studio Ollama 대비 9~22% 느림 (동일 하드웨어)
- CPU 단독: 2~8 tok/s (60배 차이)
- LLM API 가격 **1년 새 80% 인하** → 로컬 vs 클라우드 ROI 재계산 필요

### 시장 트렌드
- "Running local LLMs feels normal in 2026" (사용자 mainstream화)
- 사용량 기반 SaaS 44% (2023의 27%에서 상승) → 가시화 수요 증가
- 모델 간 성능 격차 사상 최소 → "어떤 모델 쓸지" 선택 통증 ↑

## 1.3 타깃 사용자 세분화

### Tier A1 — 클라우드 파워유저 (월 $100+ 지출)
- Claude Code + Codex 병행
- AI 코딩 일 6시간+
- 글로벌 추정: **500k~1.5M명**
- 페르소나: 인디 해커, AI 컨설턴트, 시리어스 빌더

### Tier B1 — 로컬 LLM 시리어스 유저
- 전용 GPU 보유 (RTX 3090/4090, M2/M3 Pro/Max, AMD 7900XT 등)
- 3개 이상 모델 정기 사용
- 글로벌 추정: **200k~600k명**
- 페르소나: ML 엔지니어, 프라이버시 의식자, 비용 의식자, 연구자

### Tier AB1 — 하이브리드 (가장 가치 큼)
- 클라우드 + 로컬 병행
- 양쪽 통증 동시 보유
- A1∩B1 중첩 추정: **100k~300k명**
- **WTP 가장 강함**

### Tier 2 — 1년 내 확장
- 클라우드만 일반 사용자: 3~6M
- 로컬 캐주얼 사용자: 500k~1.5M

## 1.4 시장 사이즈

### TAM (Total)
- 클라우드 측: 3~6M × $15 × 12 = **$540M~$1.08B**
- 로컬 측: 1~3M × $15 × 12 = **$180M~$540M**
- **합계 TAM: $720M~$1.62B**

### SAM (Serviceable)
- 클라우드 파워유저 + 로컬 시리어스 유저 + 하이브리드
- 약 800k~2.4M명
- × $18/월 (혼합 ARPU) × 12
- **$173M~$518M**

### SOM (3년 현실 캐치)
- SAM의 0.5~2%
- 4,000~48,000 유료 사용자
- × $18/월 × 12
- **$864k~$10.4M ARR**

기존 (클라우드 단독) 대비 **약 1.5배 시장**.

- 지속 모니터링 + 권장 + 비교 = 공백
- 하이브리드 ROI 계산 = 공백

## 1.6 진입 시점 정당성

### 왜 지금
1. **로컬 LLM mainstream화** — 2026년 "normal"
2. **클라우드 API 80% 인하** → 로컬 vs 클라우드 비교 수요 폭증
3. **모델 성능 평준화** → 선택 통증 ↑ (어느 게 빠르고 가성비 좋은지)
4. **하이브리드 사용 보편화** — 시리어스 빌더는 양쪽 다 씀
5. **사용량 기반 빌링 확산** → 가시화 도구 카테고리 정착

### 너무 늦지 않은 이유
- OSS 벤치마크 도구는 모두 **단발 CLI** → UX 공백
- 엔터프라이즈 옵저버빌리티는 **너무 무거움** → 인디 공백
- 통합 도구는 아무도 안 함 → first-mover

→ 12~24개월 윈도우 추정.
